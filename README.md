## Olá, eu sou Danielly Egito!

- Estudante de Sistemas de Informação (UFRPE) e Tutora de Tecnologia no CESAR, atuando em projetos educacionais com tecnologia, com ênfase no apoio à aprendizagem de programação para estudantes do ensino médio da rede pública de Pernambuco.

- Atuo como tutora auxiliando alunos no entendimento de lógica de programação, Python, JavaScript e conceitos de desenvolvimento web, além da elaboração e aplicação de atividades práticas e da orientação em boas práticas de desenvolvimento.

- Possuo experiência com desenvolvimento web em ReactJS e desenvolvimento mobile com Flutter, desenvolvida durante minha atuação como Analista de Projetos na Seed a Bit Tecnologia, onde participeino processo de concepção, planejamento e implementação de produtos digitais.

- Atuei como Estagiária em Agile (Scrum), colaborando com times de desenvolvimento na organização de demandas, documentação de projetos, facilitação de cerimônias ágeis e acompanhamento de sprints, utilizando ferramentas do ambiente Atlassian (Jira e Confluence).

- Tecnologias e áreas de interesse:
Python | JavaScript | React | Flutter | Desenvolvimento Web | Metodologias Ágeis | Educação em Tecnologia
